# django-rest
Restful-API development by Django rest framework

## Environment

### Ubuntu 20.04.1

```
sudo apt install python3-pip
sudo apt install python3-venv
```

### Local development

- Create venv
    ```
    python3 -m venv venv
    ```

- Install project dependencies
    ```
    pip install -r requirements.txt
    ```

- Build app docker image
    ```
    docker build .
    ```
